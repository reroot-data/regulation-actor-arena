# regulation-actor-arena

The project can be cloned to the computer and run locally.

Execute the following to run the Django server locally
```
  python manage.py runserver
```

The API can then be consulted at localhost:8000/api/ and the admin at localhost:8000/admin/.
To log in, the username 'applia' and password 'europe' need to be provided.

Please note that it is not possible to rerun the analysis, as the credentials for the Google Cloud Service Account are not included.
